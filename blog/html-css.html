Positioning: Relative, Absolute and Fixed - Crash Course

Introduction

The default position of HTML elements is ‘static’. This position cannot be moved even when using properties like “left: 10px; bottom: 40px;”. Therefore, an HTML element must first be told a specific position so that the element can be moved around.

Three possible positions are relative, absolute and fixed.

The example above shows how despite the container (purple dashed box) being told to move 200px from the left and 100px from the top, the container is not moving.

Now, once the arbitrary position of relative is given to this, it is able to move, as per the example below.

Absolute Position

position: absolute;

The property value of ‘absolute’ essentials removes an element from the document. No space is created for it and other items on the file behave as though this does not exist.

The absolute value will first try to move within its parent’s position. If its parent’s position has not been specified, it will keep looking for this in its nested ancestors. If none of them have specified a position, then the element will be positioned relative to the HTML document.

The example above shows how the pink box has moved 200px from the bottom of its parent container. Note how the remaining box behave as completely independently from the pink one, as though it did not exist.

If its container did not have any specified position, then the pink box would move 200px from the bottom of the HTML document, as per the example below.

Relative Position

position: relative;

The property value of relative moves the element relative to its original position. It also keeps the element in the normal flow of the document, as opposed to the absolute property value which removes it.

If the blue box started on the position as per the image on the left, then the code on the right gives the output right-adjacent image.

This is helpful when wanting to keep an element within the natural flow of the document.

Fixed Position

position: fixed;

The fixed property value works similarly to the absolute one in that it is also ‘removed’ from the document. However, it only ever works relative to the entire HTML document, not its parents’ position as reference. Moreover, when scrolling, it stays on your screen as you scroll.

This can be used when creating banners.

Pushing Elements Back

If you want to push an element behind others, the declaration below can be used;

z-index: -1;

All elements are 0 by default, so to push an item back, the -1 value is used.
